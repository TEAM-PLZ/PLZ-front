## 소개

### 하고 싶은 말이 있지만 직접 말하기 힘드신가요?

<p>➡<b>플리즈</b>에서는 익명성을 보장합니다!</p>

### 메시지로는 내 감정을 제대로 표현하기 힘드신가요?

<p>➡<b>플리즈</b>에서 음악과 메시지로 감정을 전달하세요!</p>

### 매일 듣는 플레이리스트는 이제 지겹나요?

<p>➡<b>플리즈</b>에서 주변사람들이 나를 위해 만든 플레이리스트를 들어보세요!</p>

<br />

LP를 선물받아 플레이리스트를 채워보아요✨
